# Changelog

## [0.2.0] - 2026-07-06

### Changed

- Ícones PNG com fundo transparente (script `icons:transparent` com sharp)
- Paleta Sonic Dark e Sonic Light ajustada para melhor distinção em protanopia/deuteranopia
- Vermelho puro substituído por laranja coral; verde puro por teal/ciano
- Reforço de `fontStyle` (bold, italic, underline) em tokens semânticos
- Cores de workbench, git decorations e terminal ANSI alinhadas à nova paleta

## [0.1.0] - 2026-07-06

### Added

- Sonic Dark color theme
- Sonic Light color theme
- Sonic Icons file icon theme com mapeamentos temáticos para extensões comuns
